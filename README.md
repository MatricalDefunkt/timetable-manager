# Timetable Manager Frontend

## Pre-requisites

-   Bunjs https://bun.sh/
-   Sanity
-   Common Sense
-   A good sense of humour
-   A brain

## Installation

1. Unzip the folder
2. Open the folder in the IDE of your choice
3. Open a terminal and run `bun install`
4. Run `bun dev` to start the development server
